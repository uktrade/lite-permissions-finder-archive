You may need an outward processing trade licence


You need a licence to import certain textiles and clothing made in Belarus. As you are sending textiles to Belarus for processing before being returned to the UK, known as outward trade processing, more favourable import quotas may apply.

Read about outward trade processing limits for Belarus.

You can apply for an import licence using the Import Case Management System (ICMS). Log in to or register for ICMS.

Read about how to apply for an outward processing trade licence using ICMS.

You can contact the Import Licensing Branch by email enquiries.ilb@bis.gsi.gov.uk or by phone on 01642 364333.
