Get an import licence using ICMS


You need a licence to import certain iron and steel products manufactured in Kazakhstan.

Read about restrictions on imports of iron and steel from Kazakhstan.

You can apply for an import licence using the Import Case Management System (ICMS). Log in to or register for ICMS.

Read about how to apply for an iron and steel licence using ICMS.

You can contact the Import Licensing Branch by email enquiries.ilb@bis.gsi.gov.uk or by phone on 01642 364333.
