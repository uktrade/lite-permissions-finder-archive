You are likely to need an import licence


You can apply for an import licence for firearms and ammunition by using the Import Case Management System (ICMS). Log in to or register for ICMS.

Read about how to apply for a firearms licence using ICMS.

You can import single-shot rifles and shotguns under an open import licence. Read about how to apply for an open import licence using ICMS.

You can contact the Import Licensing Branch by email enquiries.ilb@bis.gsi.gov.uk or by phone on 01642 364333.
